## Исследование для проекта WID — digital-продукт для Fashion-индустрии (цифровой гардероб).

#### AB-эксперимент
1. Нужно проанализировать результаты проведенного AB-теста. 
Задача была смоделировать синтетический A/B-тест и дизайн эксперимента на предоставленных данных:
 - определить цель
 - подготовить гипотезу
 - выбрать метрику
 - проанализировать ход эксперимента
 - посчитать изменение метрики в двух группах
 - подобрать корректный тест и определить значимость изменений

**Выводы** 

Аудитория была разделена на две группы. В контрольной группе A 363 пользователя, в тестовой - 437 пользователей. В качестве ключевой метрики была выбрана доля регистраций.

Выборки независимы, но различны по размеру. Предпосылка о достаточном размере выборок выполняется.

По результатам проведенного эксперимента выяснили, что доля успешных сессий в группе A = 34.71%, а в группе B = 41.87%. Lift конверсии в регистрацию в тестовой группе составили +20.64%. Прирост конверсии в тестовой группе составило +7.17 п.п. На основе проведенного Z-тест пропорций, pvalue составил 0.0191 и меньше уровня значимости, из чего следует вывод, что изменения статистически значимы.

Гипотеза о том, что регистрация по Яндекс ID положительно влияет на долю регистраций в рамках выборки нашла свое подтверждение
